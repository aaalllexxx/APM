# Changelog — APM v2.0

## [2.0.0] — 2026-03-03

### Новые команды
- `apm goto` — переход в директорию проекта (keyboard выбор)
- `apm list` — просмотр зарегистрированных проектов с проверкой ✓/✗
- `apm unregister` — удаление проекта из реестра (без удаления файлов)

### Улучшения
- Группированный help (`apm help`) с цветами через rich (Проект/Навигация/Модули/Прочее)
- `apm develop screen` — генерация Screen-класса + HTML-шаблона
- Реестр проектов — `apm create` и `apm init` автоматически регистрируют проект
- Конфигуратор `routers` — выбор между авто и ручной конфигурацией
- `apm config` загружает существующие значения из config.json
- `setup.bat` исключает `__pycache__`, `.git`, `.pyc` при копировании
- `apm.bat` поддерживает `cd` после `apm goto`

### Рефакторинг
- `register_project()` вынесена в `helpers.py` (убрано дублирование из create.py/init.py)
- Проектные модули проверяют наличие `.apm` перед работой
- Кросс-платформенный ввод через `readchar` (вместо `keyboard`)
- Обработка ошибок во всех модулях

### Исправления
- Config: Ctrl+S сохранить, ESC выйти (Windows); s/q (Linux)
